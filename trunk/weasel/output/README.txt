安裝輸入法

本品適用於 Windows XP,  32/64位 Windows 7

欲將【小狼毫】註冊到繁體中文（臺灣）鍵盤佈局，
請通過命令行介面啓動安裝程序，並指定選項 /T ：

> weasel-x.x.x-installer.exe /T

----------------------------------------------------------
使用輸入法

選取輸入法通知區的【中】字樣圖標，開始用小狼毫寫字。
通過快捷鍵 Ctrl+` 或 F4 呼出方案選單、切換輸入方式。

----------------------------------------------------------
定製輸入法

右擊托盤圖標，快速訪問配置管理工具。
修改配置文件，於「重新載入」後生效；
修改詞庫，通過「部署方案」生效。
用戶配置文件位於 %AppData%\Rime 調教Rime輸入法你會常用到。
爲免自定義內容會在輸入法升級後丟失，請勿修改安裝目錄中的
文件；如需定製預設輸入方案，應在用戶配置目錄重命名該方案。

  * 增設Shift鍵切換中／西文模式

  * 用C++重寫核心算法（階段成果）
  * 將輸入法介面從前端遷移到後臺服務進程
  * 兼容64位系統

小狼毫 0.1 ~ 0.3
  * 以Python開發的實驗版本
  * 獨創「拼寫運算」技術
