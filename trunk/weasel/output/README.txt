# 說明書

----------------------------------------------------------
【小狼毫】

基於 中州韻輸入法引擎／Rime Input Method Engine

http://code.google.com/p/rimeime/

中州式恕堂 版權所無

----------------------------------------------------------
安裝輸入法

本品適用於 Windows XP, Windows 7

----------------------------------------------------------

欲將【小狼毫】註冊為繁體中文鍵盤佈局，請手動執行安裝腳本，
並指定命令行選項 /t ：

> install.bat /t


修改介面風格，請編輯配置文件 %AppData%\Rime\weasel.yaml

----------------------------------------------------------
使用輸入法

選取輸入法通知區的【中】字樣圖標，開始用小狼毫寫字

----------------------------------------------------------
更新日誌

小狼毫 0.9
  * 用C++重寫核心算法（階段成果）
  * 將輸入法介面從前端遷移到後臺服務進程
  * 兼容64位系統

小狼毫 0.3
  * 核心算法升級為 Rime.py 0.3，仍兼容 ibus-zime 3.x 數據庫
  * 交互模式更貼近主流拼音輸入法，支持兩種輸入風格
  * 修復BUG若干

小狼毫 0.2
  * 增加安裝為繁體中文輸入法的選項
  * 可通過註冊表設定介面字體、字號
  * 更新核心算法 3.6，包括以下功能升級：
    - 重寫了數據庫管理工具，提供刪除輸入方案、
      導入、導出用戶資料的功能
    - 改善安裝輸入方案速度慢的狀況
    - 改進組句算法執行效率、調頻策略、根據上文調整候選排序
    - `鍵進入西文模式、無候選詞時可輸入網址、郵箱、浮點數
      （僅限於roman／羅馬字解析方式）
  * 推介「速成」、「吳語（老派）」輸入方案

小狼毫 0.1

----------------------------------------------------------
問題反饋

發現程序有BUG，或使用不便之處，請到Rime項目網站反饋：
http://code.google.com/p/rimeime/issues/list

已知問題

  * 安裝輸入方案過程中，輸入法暫時不可使用
  * 不支持設置為系統默認鍵盤佈局
  * 不支持嵌入到命令行和全屏遊戲的內置輸入法介面
  * 不支持多顯示器
